# Sprint 6 — Mapa interactivo + Filtros

## Objetivo
Hacer el mapa de conocimiento interactivo y filtrable.

## Funcionalidades
1. Click en nodo → panel lateral con el concepto y 
   todas sus conexiones explicadas
2. Filtro por categoría — mostrar solo nodos de 
   una o varias categorías seleccionadas
3. Filtro por nivel de dominio — mostrar solo 
   conceptos con mastery >= N

## Harness
- Click en nodo retorna datos del concepto correcto
- Filtro por categoría reduce nodos visibles correctamente
- Filtro por dominio funciona con valor 0
- Panel lateral muestra todas las conexiones del nodo
- Mapa sin conceptos no lanza error
