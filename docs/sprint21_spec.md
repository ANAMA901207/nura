# Sprint 21 — UX y diseño

## Objetivo
Mejorar la experiencia visual y de interacción de Nura
para que sea más intuitiva, consistente y agradable.

## Items

### 1. Eliminar campo contexto opcional
- Remover el segundo input de contexto del chat
- Solo un input limpio de chat
- El contexto se captura implícitamente del perfil

### 2. Rediseñar módulo Dominar
- Layout más claro y menos confuso
- Secciones bien separadas visualmente
- Jerarquía visual clara: repaso > flashcards > perfil

### 3. Mensajes motivacionales como toast/popup
- Reemplazar banner estático por toast flotante
- Aparece 2-3 segundos y desaparece
- Posición: esquina inferior derecha
- Estilo: sutil, oscuro, con ícono de estrella

### 4. Nombres consistentes con identidad Nura
- Revisar todos los textos de la UI
- "Ver modo mapa" → "Explorar constelación"
- Cualquier referencia a "chat" → "Descubrir"
- Términos técnicos visibles → lenguaje amigable

### 5. Editar conceptos más visible
- Botón editar siempre visible en tarjeta
- No escondido en expander
- Inline editing directo en la tarjeta

### 6. Click en nodo filtra mapa
- Pendiente del Sprint 20
- Click directo en nodo → filtra automáticamente
- Sin necesidad de selectbox

## Harness
- Campo contexto no aparece en UI
- Módulo Dominar tiene secciones claramente separadas
- Toast aparece y desaparece sin bloquear UI
- Nombres consistentes en toda la app
- Botón editar visible sin expander
- Click en nodo activa filtro
